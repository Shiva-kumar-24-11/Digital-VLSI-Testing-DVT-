# 💻 Digital VLSI Testing Lab (01CT0629)

This repository contains all Digital VLSI Testing lab experiments performed as part of my B.Tech ICT curriculum at Marwadi University.

The experiments focus on fault detection, design-for-testability (DFT), and verification of digital circuits using Verilog and simulation tools.

---

## 📌 Objectives

- Understand different fault models in digital circuits
- Analyze stuck-at faults (SA0 & SA1)
- Implement combinational and sequential circuits
- Learn Automatic Test Pattern Generation (ATPG)
- Study Design for Testability (DFT) techniques
- Perform memory testing using March algorithms
- Implement BIST (Built-In Self-Test) architectures
